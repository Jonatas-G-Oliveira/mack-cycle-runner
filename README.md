# Bark Beat | Jogo de ritmo focado em reabilitação

## Como jogar<br>

 - Baixar o pacote e descompactar<br>
 - Na pasta { jogo } execute o comando<br>
 - ``` ./ gradlew run ```
---
## Teclas
- Seta pra cima - Lane de cima
- Seta pra baixa - Lane de baixo
- Espaço - Acertar disco
- ESC - Voltar ao menu
